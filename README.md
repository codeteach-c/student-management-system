# Student Management System in C

A simple console-based application in C to manage student records.  
This program uses file handling to store student data permanently and provides a menu-driven interface for easy use.

## Features

- Add new student records  
- View all students  
- Search for a student by ID  
- Update student information  
- Delete student records  

## Language & Tools Used

- Language: C  
- Compiler: GCC (Windows/Linux)  
- Platform: Console Application
